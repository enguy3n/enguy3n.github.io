APPLICATIONS

GREENCHALLENGE
TOOLS: HTML, CSS, Javascript, Bootstrap, EJS, ExpressJS, NodeJS, PostgreSQL, Docker
FINISHED: May 4, 2023

GreenChallenge was created as a final project for CSCI 3308: Software Development and Tools at CU Boulder. It is a social media application where users could log activities to measure their carbon emissions and compete against other users to have a more positive footprint on the environment. I was responsible for UI and graphics of the application, alongside creating several site protocols. I also spent much of my time on the project helping others manage Git and debug their code, alongside helping the team manage deadlines and meetings.

NOTE: Because this is not currently being hosted on a server, this links to a demo rather than the actual application.


FREYJA SIMULATOR
TOOLS: HTML, CSS, Javascript
CREATED: March 18, 2023

Freyja Simulator is a retelling of Hyndluljóð, an old Norse poem, with some added story, made partially as a challenge to myself to create an application in a single night, including writing its story, designing its UI, and programming all its functionality, though minor typo fixes were made in the following days. Its functional design was inspired by text-based games, with its appearance made simple to focus the text.


PIXEL ART CREATOR
TOOLS: HTML, CSS, Javascript
FINISHED: December 2, 2022

I created a mini pixel art creator as a final project in ATLS 2200: Web, a class at CU Boulder that focused on basics of HTML, CSS, and Javascript. The application includes a wide variety of features, including custom color picking, an eyedropper tool, color saving, and dark/light mode.


GRAPHIC DESIGN

POSTER MAILER: COLORADO SYMPHONY 2023 CLASSICS
TOOLS: Adobe Illustrator, Adobe Indesign
FINISHED: April 10, 2023

This poster was created for ATLS 2300: Text at CU Boulder. The assignment was to create a poster mailer for the Colorado Symphony's 2023 Classics season performances, using a given set of text. The assignment included the additional constraints that only 2 fonts of a given set of fonts could be used, only pantone colors could be used, and the poster had to be a certain size. Given constraints, this was an exercise in working with client-specified constraints while managing to effectively display information anyway.


SPECIMEN TYPE POSTER: ROC GROTESK
TOOLS: Adobe Illustrator
FINISHED: February 26, 2023

This poster was created for ATLS 2300: Text at CU Boulder. The assignment was to pick a font with multiple styles (bold, italic, condensed) and display them in some way in the poster. The goal was to highlight the glyphs themselves through effective arrangement of text, as opposed to distracting a viewer with graphics.
